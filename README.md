# Hotel booking system
Приложение должно обрабатывать следующие запросы пользователя:
1. Создание отеля для заселения.
2. Запрос на заселение пользователя с гостями с помощью консоли или написанного письма от гостей.
3. Продление бронирования.
4. Вывод информации об отелях и постояльцах.
5. Удаление информации об отеле и постояльцах.
